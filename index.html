    SWTOR Analyzer and Parser

    Project maintained by TeamSWAP Hosted on GitHub Pages — Theme by mattgraham

    SWAP (Star Wars Analyzer & Parser) is a new open-source Star Wars: The Old Republic combat log parser written in the Python programming language.

    Download SWAP v0.8.4e (32-bit, Installer)

    If you know what you're doing, look below for more options.

    Features

    • 100% open-source: You can delve into our code, change stuff, send it back to us, or fork it! (Note: SWAP server source-code not available yet.)
    • P2P raids: SWAP doesn't synchronize raids using our central servers, instead we use NAT traversal to establish a direct connection to one of the raid members. If that's not possible we use our servers as a relay to that raid member.
    • Low CPU footprint: Lower idle parsing CPU usage than MOX parser.
    • Overlays: Our overlays are always on top, have no title bar, can be dragged, clicked and resized, all without losing focus on the game. You can even enable "Click-through Overlays"
